ðŸ“Œ What are Constructors?

A constructor is a special function used to create and initialize objects. It acts as a blueprint for creating multiple objects with the same structure and behavior.

ðŸ”§ The Old Way (Function Constructors)

Using Traditional Function Constructor

Before ES6, constructors were created using regular functions with the new keyword.

function Person(name, age) {
    this.name = name;
    this.age = age;
}

const user = new Person("Alice", 25);

Note: Constructor function names are capitalized by convention.

âœ¨ The Modern Way (ES6 Classes)

Using Class Syntax (Recommended)

ES6 introduced the class syntax, which is cleaner and more intuitive. It's syntactic sugar over traditional constructors.

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(`Hi, I'm ${this.name}`);
    }
}

const user = new Person("Alice", 25);
user.greet();

Advantages: Cleaner syntax, built-in method support, better readability.

ðŸŽ¯ Real-World Use Cases

1. Creating Multiple Similar Objects

Perfect for creating instances with the same structure.

class Book {
    constructor(title, author, pages) {
        this.title = title;
        this.author = author;
        this.pages = pages;
        this.isRead = false;
    }
}

const book1 = new Book("1984", "George Orwell", 328);
const book2 = new Book("To Kill a Mockingbird", "Harper Lee", 281);

2. Encapsulating Data & Behavior

Combine related data and methods together.

class BankAccount {
    constructor(owner, balance) {
        this.owner = owner;
        this.balance = balance;
    }

    deposit(amount) {
        this.balance += amount;
    }

    withdraw(amount) {
        if (amount <= this.balance) {
            this.balance -= amount;
        }
    }
}

3. Using Prototypes for Efficiency

Methods defined in a class are stored in the prototype, saving memory.

class Car {
    constructor(make, model) {
        this.make = make;
        this.model = model;
        this.speed = 0;
    }

    accelerate(amount) {
        this.speed += amount;
    }

    brake() {
        this.speed = 0;
    }
}
